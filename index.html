Course Structure

Welcome to the react starter course! The purpose of this course is to get you comfortable working in an advanced front-end environment quickly. By the end of this course, you should:

  • be familiar with react
  • understand how to think about building UIs with components
  • understand when and why to use state
  • have experience building a number of components and small apps

Because there's no better way to learn than doing, this course is project-focused. You will be introduced to react concepts and new javascript features as they are needed to accomplish projects. Each project description should list out the topics that are needed to successfully complete the project. Any react topics covered leading up to a project are specifcally needed for that project. Once we've covered an ES6 topic, you should try to use your new knowledge in all of the projects that follow.
